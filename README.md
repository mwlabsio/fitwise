# Fitwise

A mobile-first fitness planner and workout tracker built as a single HTML file.

## Features
- **Weekly planner** — structured schedule across Strength, Cardio/Rowing, and Yoga/Pilates
- **Exercise library** — 30 exercises with sub-focus tags and optional video links
- **Session logging** — step through all exercises in a day, log sets, reps, and weight
- **Last session tracking** — automatically shows your previous weight & reps as a reference
- **Progress history** — full session log with per-exercise summaries

## Default weekly schedule
| Day | Focus |
|-----|-------|
| Mon | Strength · Upper |
| Tue | Strength · Lower |
| Wed | Cardio / Rowing |
| Thu | Strength · Full Body |
| Fri | Cardio / Rowing |
| Sat | Yoga / Pilates |
| Sun | Rest |

## Usage
Just open `fitwise.html` in any browser — no server, no install, no login needed.
Data saves to localStorage. Firebase auth can be enabled for cross-device sync.
